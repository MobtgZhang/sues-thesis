# 上海工程技术大学硕士学位论文latex模板
这是我根据学校研究生处给的docx硕士学位论文模板撰写的latex学位论文模板，欢迎大家使用。硕士学位论文模板当中包含有以下三个部分：
+ 开题报告模板
+ 学位论文模板
+ 答辩PPT模板

其中答辩PPT模板我分为三个阶段的模板，分别表示开题答辩、中期答辩和最终答辩，细微调整了一些模板，以供大家参考和使用。

如果在编译过程中遇到什么问题，可以在项目`issue`中提出，希望共同维护硕士学位论文模板。

## 使用方法

文件目录如下所示
```bash
 
┌─imgs
│  gbt7714-2005.bst
│  logo.png
│  paper.tex
│  README.md
│  refers.bib
│  report.tex
│  sues-thsis.cls
```
